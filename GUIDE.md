# 🎰 Slot Game Data Extractor - 배포 및 사용 가이드

## 📦 배포 필수 파일

아래 **4개 파일**을 하나의 폴더에 넣어 전달해 주세요.

| # | 파일명 | 설명 |
|---|---|---|
| 1 | **`start.bat`** | 🚀 시작 파일. 더블클릭만 하면 가상환경 생성 → 패키지 설치 → 프로그램 실행까지 전자동으로 처리됩니다. |
| 2 | **`main_app.py`** | 🖥️ GUI 프로그램 본체. 비디오 플레이어, 데이터 테이블, 그래프 탭 등 UI 전체가 포함되어 있습니다. |
| 3 | **`extractor.py`** | 🔍 핵심 감지 엔진. EasyOCR로 영상의 Balance/Win 값을 분석·추출하는 백엔드 로직입니다. |
| 4 | **`requirements.txt`** | 📋 의존성 목록. `start.bat`이 이 파일을 읽고 필요한 라이브러리를 자동 설치합니다. |

> **참고**: 테스트용 영상 파일이 있다면 함께 전달하면 좋지만, 받는 분이 직접 영상을 갖고 있다면 보내지 않아도 됩니다.

## 🖥️ 사전 요구사항

1. **Python 3.8 이상** 설치
   - 설치 시 반드시 **"Add Python to PATH"** 체크박스를 선택해 주세요.
   - 다운로드: https://www.python.org/downloads/
2. **(권장) NVIDIA GPU + CUDA 지원 드라이버**
   - GPU가 있으면 감지 속도가 획기적으로 빨라집니다.
   - GPU가 없어도 CPU 모드로 자동 동작하므로 사용에는 문제없습니다.

---

## 🚀 실행 방법

1. 전달받은 4개 파일을 원하는 폴더에 넣습니다.
2. **`start.bat`을 더블클릭**합니다.
3. 첫 실행 시 가상환경 생성과 패키지 설치가 진행됩니다. (약 5~10분 소요)
4. 설치가 완료되면 프로그램이 자동으로 실행됩니다.
5. 이후 실행부터는 바로 프로그램이 열립니다.

## 📖 사용 방법

### 기본 흐름
```
📂 Select Video → 🎯 Select ROI → ▶ Start → 💾 Export
```

1. **📂 Select Video**: 분석할 슬롯 게임 영상 파일을 선택합니다.
2. **🎯 Select ROI**: Balance 영역과 Win 영역을 순서대로 드래그하여 지정합니다.
3. **▶ Start**: 감지를 시작합니다. (Start 입력칸에 시간을 지정하면 해당 시점부터 시작)
4. **💾 Export**: 결과를 엑셀 파일로 저장합니다.

### 추가 기능
- **⏹ Stop**: 진행 중인 감지를 중단합니다.
- **🔄 Reset**: 전체 초기화 후 새 영상을 분석할 수 있습니다.
- **Go 버튼**: 데이터 테이블의 시간 열에 있는 Go 버튼을 누르면 영상이 해당 시점으로 이동합니다.
- **📈 Graph 탭**: X축/Y축을 선택하여 데이터를 그래프로 시각화할 수 있습니다.
- **Start 시간 입력**: `hh:mm:ss` 형식으로 시작 시간을 지정하면 해당 시점부터 감지합니다.
